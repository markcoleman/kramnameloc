#[5 Things Every Developer Needs to Know About Design](http://schedule.sxsw.com/2014/events/event_IAP27303)
After stuffing my face I needed to rush over to the Radisson for my first full session.
![Silvrback blog image]({{ '/assets/blog-images/2014-03-09-sxsw-14-day-1/03-img_1901_large.jpg' | relative_url }})
A few take aways from this session notes...

*Attractive things work better.*

 - Layout - order, variety, contrast, symmetry, tension, balance (two different objects with the same weight), scale, (you don't have to make big to increase scale), texture, space. Stability body content. Excitement header
Golden ratio, 
 - Using imagery, better to show than tell, leading the eye, pay attention to direction. Use icons and shapes, round things are easier to look at instead of sharp edges.
 - Color, use to emphasis importance, not to just decorate a page, understand color intensity. 
 - Using Typography, serif fonts lead into each letter making it easier to read, sans good for adding emphasis very distinct letters. 
Serif for headings (slab, Rockwell), sans for body copy. Limit to no more than three fonts. 
 - Feedback, have others help you.  [Bēhance](http://www.behance.net) allows for new feedback and track some revisions.  

#[Rich Web Experiences & the Future of the Web](http://schedule.sxsw.com/2014/events/event_IAP27377)
This session turned out to be a crash course of all things new that are in the web.

 - [Clip paths](https://developer.mozilla.org/en-US/docs/Web/CSS/clip-path), can transition and alter interfaces. Revel and hide content
 - [Mask images](http://tympanus.net/codrops/2013/12/02/techniques-for-creating-textured-text/), allows for textures on top of content
 - Blend modes, background image blend modes with multiple background images. (Experimental)
 - Filters, saturate, contrast, blur, etc. filters can be stacked
 - Vector, images sag a few libraries noted are Raphaël.js (compatibility) d3.js (graph and dats), snap.js (new modern svg)
 - 2d canvas,  easel.js, processing.js, pixi.js
 - Voice, API for interesting navigation options.
 - Webgl, a few libraries noted are three.js, Babylon.js, pixi.js
 - Video, 
 - Sprite Sheets, combine images lower requests
 - Semantics, the mark up and the text in the document
 - WebRTC, real time audio video chat
This talk was ok, but was such a crash course in all things new and exciting it was hard for me to focus on a particular aspect and how I could utilize this in my day job.
